HTML Básico

HTML

HTML (Lenguaje de Marcado de Hipertexto) es el lenguaje de marcado estándar para crear páginas web. Aquí hay algunas etiquetas esenciales:
  • <html>: Envuelve todo el contenido de la página.
  • <head>: Contiene meta información y enlaces a recursos externos.
  • <title>: Define el título de la página.
  • <body>: Contiene el contenido visible de la página.
  • <h1> a <h6>: Encabezados.
  • <p>: Párrafos de texto.
  • <a>: Enlaces.
  • <img>: Imágenes.

Estilos CSS para Mejorar la Página

Utiliza CSS (Hojas de Estilo en Cascada) para mejorar el diseño y la presentación de tu página web. Aquí hay algunos consejos:

  • Define reglas de estilo en un archivo separado (Styles2.css).
  • Utiliza selectores para aplicar estilos a elementos específicos.
  • Agrega márgenes, rellenos y bordes para mejorar la legibilidad.
  • Experimenta con colores y fuentes para una apariencia atractiva.
  • Considera la creación de una barra de navegación con enlaces estilizados.
